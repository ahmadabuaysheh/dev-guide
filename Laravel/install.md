# Install laravel


```
composer global require laravel/installer
```

Add the following line to ~/.bashrc if you use normal shell, or to ~/.zshrc if you use ```zsh```

```Shell
export PATH="$HOME/.config/composer/vendor/bin:$PATH"
```

source the file
```Shell
source ~/.bashrc
```
**OR**
```Shell
source ~/.zshrc
```
